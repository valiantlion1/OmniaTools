# Omnia Caption — Product Definition (V1)

## Purpose
Omnia Caption helps creators quickly generate, edit, and export clean captions for short-form videos.

## Target User
- Solo creators and small content teams publishing short videos.
- Users who need fast caption output without complex editing workflows.

## Core Problem
Manual captioning is slow and repetitive. Existing tools can be overcomplicated for basic short-form caption workflows.

## V1 Scope
1. **Caption generation from uploaded video audio**
   - Produce a time-aligned caption draft.
2. **Basic caption editing**
   - Edit caption text line by line.
   - Adjust timing for individual caption blocks.
3. **Simple style controls**
   - Choose from a small set of readable preset styles.
4. **Export support**
   - Export caption file (e.g., SRT).
   - Export a captioned video preview output.

## Out of Scope (V1)
- Multi-language translation.
- Team collaboration and comments.
- Brand kits, advanced animation, or template marketplace.
- Long-form video production workflows.
- Public API and third-party integrations.

## Success Criteria (Early)
- A creator can go from upload to export in a short, understandable flow.
- Caption draft quality is good enough that only light edits are needed for most clips.
