# Omnia Caption — V1 Scope

## V1 Goal
Deliver a first usable caption workflow that helps creators produce and refine caption drafts quickly.

## In Scope
1. **Input brief**
   - Simple prompt/input describing content context.
2. **Caption draft generation**
   - Produce multiple short caption options per request.
3. **Tone selection**
   - Offer a small set of tone presets (for example: professional, casual, bold).
4. **Quick regenerate**
   - Re-run generation with the same brief for alternatives.
5. **Copy-ready output**
   - Clean text output optimized for quick copy/paste.

## Out of Scope (V1)
- Native social publishing integrations
- Team collaboration/workspaces
- Saved content libraries/history management
- Advanced brand voice training
- Multi-step campaign planning

## V1 Constraints
- Keep architecture minimal and app-local at first
- Prefer straightforward UX over customization depth
- Add shared packages only when real duplication appears

## V1 Exit Criteria
- A user can enter a short brief and receive multiple usable caption drafts
- A user can switch tone and regenerate without leaving the flow
- Output is consistently easy to copy and edit
